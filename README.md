# Distrito Miami: VMware to AWS Migration

Proyecto empresarial de evaluación, migración y modernización de la plataforma e-commerce Distrito Miami desde un entorno VMware hacia Amazon Web Services.

## Descripción

Este proyecto reproduce una infraestructura empresarial on-premise basada en VMware Workstation, utilizando como caso de negocio la plataforma e-commerce de Distrito Miami.

La aplicación será desplegada inicialmente en un entorno local controlado y posteriormente será migrada hacia AWS utilizando buenas prácticas de arquitectura cloud, seguridad, automatización, alta disponibilidad y observabilidad.

> Este laboratorio utiliza una versión controlada y anonimizada de la aplicación. No se incluyen credenciales, datos personales de clientes, respaldos productivos ni información sensible.

## Objetivo general

Diseñar e implementar una estrategia completa de migración desde VMware hacia AWS.

El proyecto incluirá:

- Levantamiento del entorno actual.
- Construcción del laboratorio en VMware Workstation.
- Migración mediante AWS Application Migration Service.
- Infraestructura como código con Terraform.
- Docker.
- CI/CD con GitHub Actions.
- Application Load Balancer.
- Auto Scaling.
- Amazon RDS PostgreSQL.
- Amazon S3.
- CloudWatch y SNS.
- Seguridad, respaldos y monitoreo.
- Pruebas de carga y alta disponibilidad.
- Documentación técnica y ejecutiva.

## Caso de negocio

Distrito Miami es una empresa chilena dedicada a la venta online de ropa y accesorios.

La plataforma dispone de:

- Portal web.
- API.
- Base de datos PostgreSQL.
- Catálogo de productos.
- Usuarios.
- Sistema de pedidos.
- Imágenes de productos.
- Panel administrativo.
- Integración con medios de pago.
